# Harko Releases

Repositório público de atualizações automáticas do aplicativo **Harko**.
*Aqui são armazenados os instaladores e arquivos de atualização.*

---

## Como funciona?

1. **Atualizações automáticas**:
   O Harko verifica periodicamente este repositório para buscar novas versões.

2. **Arquivos importantes**:
   - `latest.json`: Contém informações sobre a versão mais recente e links para download.
   - Instaladores assinados (Windows/macOS).

---


*Atualizações gerenciadas via GitHub Actions.*
